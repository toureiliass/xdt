select()

select(string  $selector = '*', \DOMElement  $context = null, integer  $flag = \XDTNS::SELECT_DESTROY) : \XDTNSNodeList

Parse a string containing a CSS selector expression, match the selected elements and returns a XDTNSNodeList object containing the set of matched elements; XDTNSNodeList class implements arrayAccess and iteratorAgregate interfaces, so its objects can to be treated like an array; jQuery specific selectors (first, last, nth, eq) are also supported.

Parameters

string $selector

[optional]

A string containing the selector expression. You can provide multiples selectors by separating them with a comma (,).
\DOMElement $context

[optional]

DOM element used as the selection context.
integer $flag

[optional]

Accepted flags are:

  • XDTNS::SELECT_DESTROY end the current operation and start over a new operation.
  • XDTNS::SELECT_FILTER filter the current result set.

Returns

\XDTNSNodeList

final
Inherited from
\XDTNS
